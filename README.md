# Qwen-Chain-MoE
Enable users to run a Mixture of Experts (MoE) system with multiple Qwen expert models under low GPU memory footprint, achieving maximum functional capability.
让用户能在较小的显卡内存占用下，实现多Qwen专家模型的MoE运行，以实现最大功能.
